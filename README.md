# Telecom Churn Analysis 📊

## 🔍 Project Overview
This project analyzes customer churn using Excel to visualize key insights across various segments, including Gender, Lifecycle Stage, Payment Method, and Subscription Plan. The goal was to transform raw data into actionable business strategies to improve customer retention.

## 💡 Summary and Findings

### 1. Subscription Plans
* **42.71%** of customers on the **Month-to-Month** plan churned.
* These customers represent **88.55%** of the total churned population.
* **Observation:** The price difference between plans is small. We should encourage these users to migrate to 1-year or 2-year plans.

### 2. Customer Lifecycle
* **54.71%** of onboarding customers (**0–6 months**) churned.
* This group accounts for **38.81%** of the total churn, indicating a critical need for better engagement during the initial onboarding phase.

### 3. Payment Methods
* **45.29%** of customers using **Electronic Checks** and **19.11%** using **Mailed Checks** churned.
* Together, these manual methods represent **73.78%** of all churned customers.
* **Insight:** Because these methods require an active decision each month, they increase the likelihood of cancellation. 

---

## 🚀 Recommendations
To improve retention, the following strategies are proposed to move customers toward automatic payments and long-term plans:

* **Autopay Incentives:** Offer discounts for enabling autopay (e.g., Save $10/month).
* **Default Settings:** Set autopay as the default option within the mobile application.
* **Value-Add Bonuses:** Offer bonuses like extra data or specialized add-ons for autopay users.
* **Communication:** Clearly communicate the benefits of autopay, emphasizing safety, convenience, and the flexibility to cancel at any time.
